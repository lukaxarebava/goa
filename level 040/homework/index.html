Flex Container-ის კუთვნილებები:

display flex: გააქტიურებს flexbox განლაგებას კონტეინერში.

flex-direction განსაზღვრავს ელემენტების განლაგების მიმართულებას მთავარ ღერძზე

justify-content განსაზღვრავს ელემენტების განლაგებას მთავარ ღერძზე.

align-items განსაზღვრავს ელემენტების განლაგებას ჯვარედინი ღერძზე.

align-content განსაზღვრავს მრავალხაზოვანი ელემენტების განლაგებას ჯვარედინი ღერძზე.

Flex Item-ის კუთვნილებები:

flex-grow განსაზღვრავს, რამდენად გაიზრდება ელემენტი თავისუფალი სივრცის შესავსებად.

flex-shrink განსაზღვრავს, რამდენად შემცირდება ელემენტი სივრცის ნაკლებობისას.

flex-basis განსაზღვრავს ელემენტის საწყის ზომას.

order განსაზღვრავს ელემენტების რიგის თანმიმდევრობას.

align-self განსაზღვრავს კონკრეტული ელემენტის განლაგებას ჯვარედინი ღერძზე.

margin, padding, border, content ახსნა:

Padding შიდა სივრცე ელემენტის შიგთავსსა (content) და მის საზღვრებს (border) შორის.

Border ელემენტის გარშემო ხაზი, რომელიც მის საზღვრებს აღნიშნავს.

Content ელემენტის ძირითადი შიგთავსი, მაგალითად, ტექსტი ან სურათი.

Box Model

ეს ოთხივე კუთვნილება ქმნის CSS Box Model-ს, რომელიც განსაზღვრავს, როგორ გამოიყურება და იხსნება ელემენტი ვებ-გვერდზე.
